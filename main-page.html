What we learned

Throughout my journey in learning HTML, CSS, and JavaScript, I discovered how these three technologies work together to build a complete website. HTML taught me how to structure content, CSS showed me how to style and design it, and JavaScript introduced interactivity. Each language plays a unique role but also complements the others, making web development both logical and creative. This class gave me a clearer understanding of how websites come to life from code.

About HTMs

In HTML, I learned the importance of creating a strong foundation for a webpage. Using elements like headings, paragraphs, images, and links helped me understand how to organize content properly. I also explored semantic tags, which make websites easier to read for both users and search engines. Learning HTML showed me that good structure is the first step toward building functional and accessible websites.

About CSS

CSS allowed me to transform plain HTML into something visually appealing. I learned how to control layout, colors, fonts, and spacing, which made a big difference in presentation. Features like flexbox and grid taught me how to position elements more effectively. Through CSS, I realized how design choices impact user experience and how small changes can make a website more engaging.

About JavaScript

JavaScript was where I truly saw interactivity come into play. I learned how to make buttons respond to clicks, forms validate input, and pages update dynamically without reloading. It made me appreciate how JavaScript connects logic with design, making websites more functional and user-friendly. Overall, JavaScript tied together everything I learned in HTML and CSS, helping me build more complete and interactive projects.
